## 山东理工大学新学期思想动态调查问卷系统

### 使用方法

把代码拉到本地：

```bash
git clone https://github.com/oxyzhg/sdut-survey.git
```

进入项目文件夹：

```bash
cd sdut-survey-react
```

安装依赖：

```bash
#使用npm
npm install

#使用yarn
yarn
```

开启本地服务：

```bash
#使用npm
npm start

#使用yarn
yarn start
```

打包应用：

```bash
#使用npm
npm build

#使用yarn
yarn build
```

然后把打包后的文件 `build` 文件夹上传就行了，可以改个名字。



### 所用技术

- react16
- react-router4
- es6
- antd-mobile
